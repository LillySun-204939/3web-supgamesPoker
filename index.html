SupGames Poker

Poker game with HTML5 & nodeJS

Created by Gaël Demette / @nainterceptor

Design of app

  • MVC pattern
  • Modules
  • Graphics

About MVC pattern

  • Get HTML file content in string, and replace ##placeholders##
  • Replace placeholders if a file with same name exist
  • Require model
  • Bridge model <-> view in controller
  • Router for sockets and requests

About modules

JEEP-like (Just enough essential parts)

  • MongoDB Native
  • Sockets IO
  • => Lightweight

About graphics

  • Canvas for cards
  • Background like a table in room
